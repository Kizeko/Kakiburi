2.インストール方法
・f_feltpen11.zipを解凍して下さい。
・f_feltpen04.otfがフォントファイル本体です。

Winodwsの場合
・フォントファイルをコントロールパネルの「フォント」にコピーしてください。

Mac OS 8.1〜9.2の場合
・フォントファイルを「システムフォルダ」内の「フォント」フォルダにコピーしてください。

Mac OS X 10.1以降の場合
・フォントファイルをhome/Library/Fonts/フォルダにコピーしてください。
※homeはログインしているユーザ名になります。
※Mac OS X10.3以降であればFont Bookアプリケーションを使ってインストールすることもできます。

--------------------------------------------------------------------------------
